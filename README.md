A companion Discord bot for the virtual pet adoption website Chicken Smoothie. View pound/lost and found opening times and receive autoreminders for when they open.

## Usage

I would prefer if you don't your run own instance, but rather use the already existing instance. Head to <https://cs-pound.haruyuki.moe/> and click the button on the top right (or below the title if you're on mobile), to invite the bot to your server.

## Versioning
CS-Pound has gone through multiple iterations over the years. I have finalized a versioning scheme that uses bunny code names.

Major releases are named after different breeds of rabbits, while minor releases follow a date-based format (for example, 2026.01 for a minor update released in January 2026).

Under this scheme, previous versions of the bot are named and referenced as follows:
- [discord.py async version](https://github.com/haruyuki/cs-pound-async) - "Netherland Dwarf": The initial version of the bot, built using the async branch of discord.py in 2018. Originally created as part of my high school major project.
- [discord.py rewrite version](https://github.com/haruyuki/cs-pound-rewrite) - "Holland Lop": A rewrite of the bot using the rewrite branch of discord.py in late 2018, featuring improved code structure and new functionality introduced by the rewrite.
- [discord.js version](https://github.com/haruyuki/cs-pound-rex) - "Rex": A learning-focused rewrite in JavaScript using discord.js in 2024, also motivated by accumulated technical debt from the Holland Lop version.
- [discord.py version **(Current)**](https://github.com/haruyuki/cs-pound-lionhead) - "Lionhead": Rewritten back in Python using modern discord.py, with a focus on improved performance, maintainability, and long-term stability.

Future major releases will continue to follow the rabbit breed naming convention.
